# Styles and Appearance

Styles and Appearance of Group Button work based on CSS class. You need a style sheet to apply styles to Group Button control that is, ej.mobile.all.min.css.

## Customize styles

To change the color or design of Group Button controlâ€™s appearance, you can override default styles of Group Button control by using its class names. The following are specific class names you can use to modify styles and appearance in Group Button.

<tr>
<td>
e-m-corner-right</td><td>
This class is applied to label element of rightmost Group Button item.</td></tr>
