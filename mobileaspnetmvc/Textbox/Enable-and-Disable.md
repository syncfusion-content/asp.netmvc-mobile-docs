# Enable and Disable

Enabled property is used to enable or disable your Textbox control. The default value is true.

Refer to the following code example.

{% highlight html %}

@Html.EJMobile().TextBox("textbox_sample").WatermarkText("Textbox").Enabled(false)

{% endhighlight %}
