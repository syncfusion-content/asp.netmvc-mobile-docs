# Customize Orientation        

Customize Orientation specifies the display orientation. It can be,

1. Horizontal.
2. Vertical. 

{% highlight html %}

    @Html.EJMobile().Rating("rating_sample").Orientation(Orientation.Vertical)

{% endhighlight %}
