# Readonly

When it is set to true, you cannot edit the value in the TextArea. The default value is false.

Refer to the following code example.

{% highlight html %}

@Html.EJMobile().TextArea("mailMessage").WatermarkText("TextArea").ReadOnly(true)

{% endhighlight %}
