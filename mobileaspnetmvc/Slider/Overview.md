# Overview

The Essential JavaScript Mobile Slider widget provides support to select a value from a particular range as well as a range of value.Â The Slider has a sliding base in which the handles are moved. There are two types of the Sliders:  default slider and range slider.

## Key features

* Orientation: Slider control is displayed in horizontal or vertical direction.
* Animation: Supports for handle movement with animation.
* Range Slider: Selects a range of values.
